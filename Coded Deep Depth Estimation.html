Coded Image-Based Depth Estimation

Overview

Developed a depth estimation pipeline by simulating coded images using phase masks and training a U-Net-based neural network on monocular RGB images. The project explores the impact of coded imaging on depth estimation accuracy, evaluating models on NYUv2 and UMDCodedVO datasets.

Coded Image Generation

  • Depth Layer Segmentation – Discretized continuous depth maps into predefined layers using torch.bucketize.
  • PSF-Based Convolution – Applied Point Spread Functions (PSF) to RGB images to simulate depth-dependent blurring.
  • Multi-Layer Convolution – Designed a linear convolution model to apply varying blur intensities across different depth layers.
  • Dataset Processing – Generated coded images for UMDCodedVO-LivingRoom, DiningRoom, and NYUv2 datasets.

Depth Estimation with Deep Learning

  • Neural Network Model – Implemented a U-Net-based depth estimation model for predicting depth from coded images.
  • Loss Functions – Compared absolute error loss (L1 Loss) and frequency-based loss for improved edge sharpness.
  • Training & Optimization – Trained the model on coded and pinhole camera images, optimizing for generalization.
  • Performance Metrics – Evaluated results using:
      • Relative Absolute Error (Rel-Abs) → 0.0575 Root Mean Square Error (RMSE) → 0.0405 Inference Speed → 212.62 FPS

Challenges & solutions

  • Loss of Depth Edges in Predictions → Introduced frequency-based loss for sharper depth boundaries.
  • Computational Complexity of Convolutions → Optimized PSF application with efficient tensor operations.

Results & Learnings

  • Successfully simulated coded images using PSF-based blurring techniques.
  • Developed a U-Net model capable of depth estimation from coded images with strong accuracy.
