Project Title: Mecanum Wheel Robot

1. Overview

Built a robot using Mecanum wheels for precise and omnidirectional navigation. Integrated an optical flow sensor for localization, replacing traditional encoders, and employed EKF for real-time state estimation.

2. Background and Motivation

The project aimed to explore alternative localization techniques for robots in environments where encoder-based odometry is unreliable.

3. Implementation

Technical Approach:

  • Designed the chassis and integrated Mecanum wheels for multi-directional movement.
  • Used ZeroMQ to synchronize data from the optical flow sensor and IMU.
  • Developed Python programs (opticalflow.py, icm20948.py) for high-frequency data acquisition.

Challenges and Solutions: Overcame synchronization issues by carefully tuning the data pipeline for real-time updates.

4. Results and Outcomes

  • Achieved smooth and precise navigation in various directions.
  • Improved localization accuracy by avoiding encoder slip errors.
