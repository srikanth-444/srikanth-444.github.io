Design and Implementation of an Autonomous Combat Robot

Overview

Developed a software stack for an autonomous racing vehicle, implementing LIDAR-based localization, proportional control, and computer vision. The system successfully navigated a track, performed reverse driving, and recognized stop signs autonomously.

Software Architecture & System Integration

Localization & State Estimation

Implemented an Extended Kalman Filter (EKF) for localization, fusing:

  • Designed a modular chassis to house all components while maintaining accessibility and balance.
  • Incorporated mecanum wheels for omnidirectional movement, enabling smooth navigation.
  • Developed two weapon systems
    • Bicycle motion model for vehicle kinematics.
    • LIDAR-based Iterative Closest Point (ICP) for pose correction. Addressed uncertainty growth by tuning process and sensor noise covariances.

Navigation & Control

  • Developed a Proportional Controller (P-Controller) for lane centering and orientation correction.
  • Implemented forward and reverse driving modes, modifying control gains for stability.
  • Designed a decision-making flow based on LIDAR perception to handle turns and track boundaries.

Computer Vision for Stop Sign Detection

  • Used OpenCV-based blob detection to identify stop signs in real-time
  • Applied Gaussian blurring, color filtering, and contour detection for robustness.
  • Optimized stopping logic to reduce false positives and prevent redundant stops.

inter-process Communication & Data Flow

  • Integrated ROS-based messaging for real-time sensor data processing.
  • Optimized data flow between perception, planning, and control nodes for smooth operation.

Future Challenges & solutions

  • Advanced Localization: Incorporate IMU + Optical Flow for precise motion tracking.
  • Power Optimization: Improve efficiency with better battery management.

Results & Learnings

  • LIDAR Noise & Missing Data → Implemented filtering & adjusted scan parameters
  • EKF Localization Drift → Improved ICP-based corrections & tuned noise models.
