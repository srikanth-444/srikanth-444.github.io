Autonomous Racing with AWS DeepRacer

Graduate Advance Robotics Course, Spring 2024

Overview

This project demonstrates my skills in autonomous robotics and vehicle control. I designed and implemented a system for the AWS DeepRacer to autonomously navigate a race track. My contributions included:

  • Developing control algorithms for forward and reverse driving.
  • Implementing LiDAR-based localization using an Extended Kalman Filter (EKF).
The project involved overcoming challenges in sensor data processing, state estimation, and real-time control. It showcases my ability to apply techniques in robotics, including proportional control, Kalman filtering, and computer vision, to a complex autonomous navigation task.

Autonomous Track Navigation

The primary challenge was to develop a control system enabling the AWS DeepRacer to autonomously navigate the race track without collisions.

  • The solution involved creating a driver node with two operational modes
  • Developed Algorthim to find the points that are being intercepted by walls, and predict the robot relative postion from wall using Linear Regression
  • Implemented a proportional controller to adjust the car's position and orientation From the sensed relative position

The system allowed for autonomous navigation, but the car experienced some instability and collisions, particularly during turns.

LiDAR-based EKF for Localization

  • Developed EKF that could combine the car's dynamic model with LiDAR measurements to provide the best possible estimate of the car's location and orientation relative to its starting point
  • Bycycle Model was used for system approximation
  • ICP was used over point cloud generated by lidar giving the transformation of the robot with relative to previous position

The EKF implementation had limited accuracy, with the estimated trajectory deviating from the actual path and increasing uncertainty over time.

Reverse Driving

  • The PID Controller designed for forward driving was not suitable for reverse driving due to the car's different dynamics in reverse.
  • Implemented a separate PID controller for reverse driving, which was tuned to ensure smooth and stable operation.

Key takeaways

  • The project demonstrated the complexities of autonomous vehicle control, highlighting the importance of accurate sensor data and robust control algorithms.
  • LiDAR-based localization using EKF is challenging and sensitive to sensor noise and model inaccuracies.
  • Reverse driving requires careful tuning of control parameters to maintain stability.

Technologies Used

AWS DeepRacer, LiDAR, ROS, Python, OpenCV, SimpleICP.
