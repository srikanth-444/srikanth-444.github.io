Design and Implementation of an Autonomous Combat Robot

Overview

Developed an autonomous robot capable of real-time navigation and precise balloon-popping using advanced motion control, sensor fusion, and embedded systems. The robot seamlessly integrated mechanical, electronic, and software subsystems to achieve full autonomy.

System Integration

Mechanical Integration

  • Designed a modular chassis to house all components while maintaining accessibility and balance.
  • Incorporated mecanum wheels for omnidirectional movement, enabling smooth navigation.
  • Developed two weapon systems
    • Lance Mechanism (Close-Range): Servo-powered scissor system with an X-Acto blade.
    • Disk Shooter (Long-Range): 74,000 RPM flywheel launching limonene-coated disks.

Electronics & Sensors

  • Integrated multiple microcontrollers (Raspberry Pi & Arduino Mega) to handle high-level logic and motor control.
  • Implemented sensor fusion with
    • IMU for orientation and acceleration data.
    • Ultrasonic sensors for obstacle detection and avoidance.
    • pixy Camera for ballon detection and tracking.

Software & Control

  • Developed Python-based control algorithms for real-time decision-making.
  • Implemented custom serial communication between Raspberry Pi and Arduino for efficient data transfer.
  • Designed an autonomous navigation system with
    • Obstacle avoidance using IR and ultrasonic sensors.
    • Adaptive movement logic for tracking and engaging targets.

Future Challenges & solutions

  • Advanced Localization: Incorporate IMU + Optical Flow for precise motion tracking.
  • Power Optimization: Improve efficiency with better battery management.
  • Enhanced Path Planning: Implement Model Predictive Control (MPC) & RRT for smoother navigation.

Results & Learnings

  • Successfully developed a fully autonomous robot with integrated mechanical, electronic, and software systems.
  • Demonstrated effective real-time control, sensor fusion, and motion planning.
  • Gained hands-on experience in robotics, embedded systems, and system integration.
